Cocktail Mate

Multi-page website made with Thymeleaf and Bootstrap connected to a self-built MySQL database. Functionality built with Spring Boot MVC, full Rest API and JSON data available.

Deployed with AWS and currently only compatible with desktop. Mobile functionality is in development as well as a few other improvements.

More detailed information and future improvements can be found in the GitHub Readme file.

Customer Tracker

Another app built with Spring Boot, Thymeleaf and MySQL. The homepage displays all customers present in the database and supports full CRUD functionality to update the database in real time.

It will soon be enhanced with Spring Security to allow safe access to the database as well as to assign roles to each customer that will receive personalised newsletters based on their role.

Tide Calculator

Simple app that communicates with three different Rest APIs. Firstly, it uses the Google Maps API to support auto-complete of the location.
Then this address is sent to the Map Quest API in order to retrieve the locations latitude and longitude coordinates.
Lastly, it sends these coordinates to the World Tides API and receives the tide height at the nearest Tide Station within 100km.

Crypto-commerce

Simple Bootstrap e-commerce website where users can buy goods using Cryptocurrency.
Uses Smart Contracts written in Solidity and ran on a local Blockchain using Truffle Ganache.

Cryptocurrency Arbitrage

An app that uses the CoinMarketCap Rest API to receive all of the latest Cryptocurrency prices, which are then ran through a self-implemented Bellman Ford algorithm.

If a negative cycle is detected by the algorithm, an opportunity for arbitrage is present. The result of exchanging $1,000 into the given currencies is then displayed to the user.

Pip Bandito

A simple 2D side scroller game where the main character has to avoid incoming enemies.
Highly responsive and fluid gameplay is a must and will be built with the Godot game engine.
